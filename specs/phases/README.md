# Phase Ticket Files

This directory contains ticket files for each development phase.

## Naming Convention

`PHASE-{N}-{name}.md`

Examples:
- `PHASE-1-foundation.md`
- `PHASE-2-core-web.md`
- `PHASE-3-ai-engine.md`
- `PHASE-4-mobile.md`
- `PHASE-5-integrations.md`
- `PHASE-6-polish.md`

## File Format

Each phase file contains:

1. **Phase metadata** — Status, dates, goal
2. **Scope** — In scope / out of scope
3. **Ticket table** — ID, title, status, estimate, blocked-by
4. **Ticket details** — Full description for each ticket

## Ticket Statuses

| Status | Meaning |
|--------|---------|
| TODO | Ready to work |
| IN_PROGRESS | Currently being implemented |
| BLOCKED | Waiting on decision |
| DONE | Complete and committed |
| SKIPPED | Explicitly skipped |

## Generated By

Phase files are generated by the `/init-phase N` command.
